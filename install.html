The steps for installing the extension are as follows -
  • Open Mozilla Firefox and select Add-ons from the pull down menu at the upper right corner or use the shortcut Ctrl+Shift+A
  • Now, in the upper right corner of the Add-ons screen, click on the 'Settings' option
  • In the pull-down menu which appears, select 'Install Add-on from file' and select the file we sent you
